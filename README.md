# 🚀 YouTube Link Fetcher

> Fetch YouTube video links instantly using simple search queries.

A lightweight Python tool powered by SerpAPI that allows you to quickly retrieve YouTube video links for any topic. Built for simplicity, speed, and easy integration into automation workflows.

---

## 📖 Overview

This project helps you:
- Search YouTube using a keyword
- Extract only video links (clean output)
- Automate content discovery workflows

---

## ✨ Features

- 🔍 Simple keyword-based search
- 🎯 Outputs only YouTube links
- ⚡ Fast and lightweight
- 🧩 Easy to integrate into other tools
- 🛠 Beginner-friendly

---

## 🛠 Tech Stack

- Python
- Requests library
- SerpAPI (YouTube search engine)
